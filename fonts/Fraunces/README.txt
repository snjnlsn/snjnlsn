Fraunces Variable Font
======================

This download contains Fraunces as both variable fonts and static fonts.

Fraunces is a variable font with these axes:
  SOFT
  WONK
  opsz
  wght

This means all the styles are contained in these files:
  Fraunces-VariableFont_SOFT,WONK,opsz,wght.ttf
  Fraunces-Italic-VariableFont_SOFT,WONK,opsz,wght.ttf

Get started
-----------

1. Install the font files you want to use

2. Use your app's font picker to view the font family and all the
available styles
